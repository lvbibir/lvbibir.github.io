0 前言#

装完 wsl 后发现用户目录下的 .bashrc 文件总是无法正常读取, github 上关于此问题的 讨论 也没有比较好的解决方法

1 解决办法#

我这里取巧了一下, 在 .bash_profile 中再调用一下 .bashrc, 如下

echo >> ${HOME}/.bash_profile <<- 'EOF'
source ${HOME}/.bashrc
EOF

source ${HOME}/.bash_profile
